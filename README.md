# Python To-Do Manager with SQLite

A command line task manager written in Python with SQLite database storage.

## Features

- add tasks
- list all tasks
- mark tasks as done
- delete tasks
- persistent task storage with SQLite

## Technologies

- Python
- SQLite

## Run

```bash
python todo_app.py
```

## Example

The program allows users to manage tasks through a command line menu and stores all data in a local SQLite database.
